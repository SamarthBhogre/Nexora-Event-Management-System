# Nexora - Event Management Website

Modern, scalable event management platform built with React, Express, TypeScript, and MySQL.

## 📋 Project Overview

Nexora is a comprehensive event management system where:
- **Organizations** can create and manage events
- **Users** can browse, search, filter, and register for events
- **Admins** can oversee platform operations
- **Organizers** can manage their events and attendees

## 🎨 Design Philosophy

- **Modern**: Contemporary design patterns and animations
- **Minimal**: Clean, uncluttered interface
- **Professional**: Enterprise-grade architecture
- **Responsive**: Works seamlessly on all devices
- **Accessible**: Dark mode and accessibility support

## 🛠️ Tech Stack

### Frontend
- **React 18** with TypeScript
- **Vite** for fast development
- **Tailwind CSS** for styling
- **React Router** for navigation
- **Axios** for API communication

### Backend
- **Node.js** + **Express.js**
- **TypeScript** for type safety
- **MySQL** for data storage
- **JWT** for authentication
